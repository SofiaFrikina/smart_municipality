### Прототип веб-сервиса "Умный муниципалитет" на примере города Краснодара

Данный веб-сайт дает возможность:
- проанализировать KPI-показатели эффективности деятельности органов местного самоуправления;
- проанализировать обращения, поступающие в органы местного самоуправления;
- выявить с помощью тепловых карт слабые места города.

Виды использовавшихся графиков:
- гистограммы;
- круговая диаграмма;
- шкалы;
- карточки.
